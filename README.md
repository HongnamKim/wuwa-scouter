# 명조스카우터 (Wuwa Scouter)

> 명조(Wuthering Waves) 에코 세팅을 **근사적으로 평가**하는 웹 도구 — 에코 점수·상대 점수, 메인 옵션 추천, 빌드 비교를 한 화면에서.

**🔗 라이브 데모 — [wuwa-scouter.vercel.app](https://wuwa-scouter.vercel.app)**

"지금 내 세팅이 이론상 최대치에 얼마나 가까운가"를 가늠하기 위한 **근사 점수**를 제공합니다. 실제 데미지를 재현하지 않고, 에코·무기·버프가 영향을 주는 부분만 추린 **상대 성능 값**으로 계산합니다.

<sub>React 18 · TypeScript · Vite · Vitest · React Router</sub>

---

## ✨ 주요 기능

- **에코 세팅 점수 (딜 상승 수치)** — 무기·화음 세트·메인 에코·코스트 구성·부옵을 입력하면 개인 딜 근사치를 계산합니다.
- **상대 점수** — 두 가지 기준선 대비 내 세팅의 위치.
  - **최고점 대비**: 유효옵이 이론상 완벽하게 떴을 때(천장) 대비 %.
  - **크크작 대비**: 크리·크피 위주의 현실적 기준선 대비 % (100%가 "충분한 수준").
- **메인 조합 추천** — 코스트별 4코 메인 / 3코 조합을 최고점·크크작 기준으로 추천. 버프 상황에 따라 추천이 바뀝니다.
- **비교 화면** — 무기 교체·에코 교체 시 딜/스펙/유효옵 변화를 나란히 비교. **빌드 수정(추가 버프 조절)** 으로 즉시 반영됩니다.
- **서포터 지원 (공명 효율 우선)** — 공격력을 쓰지 않는 서포터(체력·방어력 계수)도 지원합니다.
  - **요구 공명 효율 도달을 최우선**으로 최적화 — 자기 딜을 깎더라도 요구 공효를 먼저 채우고 남는 부옵·메인으로 딜 최대화.
  - 스탯 메인은 캐릭터 **계수 스탯**을 따름(HP 계수 → HP%, 방어 계수 → 방어력%).
  - 요구 공효에 도달하지 못하는 추천 조합은 **빨간 글씨와 느낌표(❗)** 로 경고 표기.
- **파티 버프 · 추가 버프** — 파티원 편성·조건부 버프를 반영하고, 출처별(고유 스킬/무기/화음 세트/파티/기타)로 조절합니다.
- **모바일 대응 · 출시 버전 잠금 · FAQ** 내장.

## 🧮 계산 원리 (요약)

- 동작 계수(MV)처럼 에코를 바꿔도 변하지 않는 요소는 서로 상쇄되므로 계산에서 제외합니다.
- 딜은 대략 `스킬 계수 × 공격력 × (1 + 증가합) × (1 + 부스트합) × 크리`처럼 **곱 구조**로 이어집니다. 이미 큰 항을 더 올리면 효율이 떨어지고 다른 항이 유리해지므로, 버프 상황에 따라 추천 메인이 바뀝니다.
- **점수에서 의도적으로 제외**: 이상효과 딜(서리·불꽃·풍식 등), 돌파(공명 체인)로 인한 스킬 배율 변화, 부스트·방무·저무·취약 등(상대 점수에서 약분).

> 자세한 설명은 앱 내 **FAQ**(`/faq`) 또는 [`src/content/faq.md`](src/content/faq.md) 참고.

## 🚀 시작하기

```bash
npm install

npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # 타입체크(tsc --noEmit) + 프로덕션 빌드
npm run preview  # 빌드 결과 미리보기
npm test         # 테스트 (Vitest)
```

## 📁 프로젝트 구조

```
src/
├─ engine/      순수 TypeScript 계산 엔진 (점수·추천·비교, UI 비의존)
│  ├─ theory.ts    이론 최고·크크작·메인 조합·서포터 ER-우선 최적화
│  ├─ build.ts     빌드 입력 집계·공명 효율 계산
│  └─ perf.ts      통합 성능(딜 근사) 계산
├─ components/  React UI (분석·비교·목록·버프 패널 등)
├─ data/        캐릭터·무기·에코·화음 세트 데이터 (JSON)
├─ state/       상태 스토어 (localStorage 저장)
└─ content/     FAQ (Markdown)
tests/          Vitest 단위 테스트
```

계산 엔진은 UI와 분리된 순수 함수로 작성되어 있어 테스트로 검증됩니다.

## ⚠️ 참고

- **근사 도구**입니다. 절대 데미지가 아니라 같은 캐릭터의 빌드 비교용 상대 지표입니다.
- 기준선이 캐릭터마다 다르므로 **서로 다른 캐릭터의 점수를 나란히 비교하는 것은 의미가 없습니다.**
- 일부 미출시 캐릭터·정보는 인게임 스토리 기반 추정치이며 정식 출시 시 달라질 수 있습니다.
- 쿠로게임즈(KURO GAMES) 및 Wuthering Waves와 무관한 **비공식 팬 도구**입니다.
